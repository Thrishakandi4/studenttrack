About Student Tracking System

What is this Project?

The Student Tracking System is a web app that helps schools track student academics and behavior in one place. It makes it easy for teachers to manage marks, behavior records, and communicate progress with parents.

Why was it Created?

To simplify monitoring student progress and improve parent engagement through timely updates â€” removing the hassle of complicated systems.

Who is it For?

  • Teachers: Efficiently update and manage student data.
  • Parents: Receive progress reports without logging into portals.
  • Students: Stay motivated by transparent tracking and communication.

Key Features

  • Manage marks for multiple exams (Mid1, Mid2, Semester).
  • Visual performance indicators (Good, Average, Poor) with color coding.
  • Automated email notifications to parents on updates.
  • Responsive, user-friendly design with Bootstrap 5.
  • Easy record deletion with confirmation.

Technologies Used

Frontend: HTML, CSS, JavaScript, Bootstrap 5
Backend: Node.js, Express
Database: MongoDB / MySQL (whichever you use)
Email Service: Nodemailer

Benefits

  • Saves teachers time through automation.
  • Keeps parents informed easily.
  • Encourages students with visible progress tracking.
  • Scalable for future AI and analytics features.

Future Plans

  • User login with roles.
  • SMS and push notifications.
  • Analytics dashboard for teachers and admins.
  • AI-based personalized learning suggestions.
